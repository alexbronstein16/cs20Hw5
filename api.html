Soccer Team Search

Additonal Content:

This uses the TheSportsDB API to retreive soccer team information. I chose this API because it was free and I like soccer. I found this API at https://www.thesportsdb.com/documentation?

I used the searchteams.php endpoint with the paramater t (team name) which takes the result of whatever is put in the search bar. This API would be helpful for looking up basic information about soccer teams. For example, it could be used by a mobile app that displays match information to determine what stadium and competition a given team is playing in.
